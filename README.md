```

├── index.html          # 메인 페이지 (리다이렉트)

```

## 🔧 로컬 개발

1. 저장소 클론
```bash
git clone https://github.com/citron0137/citron0137.github.io.git
cd citron0137.github.io
```

2. 로컬 서버 실행
```bash
# Python 3
python -m http.server 8000

# 또는 Node.js
npx serve .
```

3. 브라우저에서 `http://localhost:8000` 접속
